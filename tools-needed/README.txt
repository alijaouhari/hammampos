TOOLS NEEDED FOR INSTALLATION
==============================

Download and place these installers in this folder before deployment:

1. Node.js (REQUIRED)
   - Download: https://nodejs.org/en/download
   - Version: LTS (Long Term Support) - currently v20.x
   - File: node-v20.x.x-x64.msi
   - Size: ~30 MB

2. AnyDesk (For Remote Support)
   - Download: https://anydesk.com/en/downloads/windows
   - File: AnyDesk.exe
   - Size: ~4 MB
   - Use: Remote support for customers

3. TeamViewer (Alternative Remote Support)
   - Download: https://www.teamviewer.com/en/download/windows/
   - File: TeamViewer_Setup.exe
   - Size: ~25 MB
   - Use: Alternative to AnyDesk

OPTIONAL TOOLS:
---------------

4. VirtualBox (For Testing)
   - Download: https://www.virtualbox.org/wiki/Downloads
   - File: VirtualBox-x.x.x-Win.exe
   - Size: ~100 MB
   - Use: VM testing before deployment

5. Windows 10 ISO (For Testing)
   - Download: https://www.microsoft.com/software-download/windows10
   - File: Win10_xxxx_English_x64.iso
   - Size: ~5 GB
   - Use: Create test VM

NOTES:
------
- Keep these installers updated
- Test installers before taking to customer
- Node.js is REQUIRED for HammamPOS to run
- Remote support tools are optional but recommended
